Html დოკუმენტის სტრუქტურა


  • < !DOCTYPE html>

    • ეს არის დოკუმენტის ტიპის დეკლარაცია.
    • ის მითითებას აძლევს ბრაუზერს რომ დოკუმენტი დაწერილია HTML5 ვერსიაზე.
    • ამ კოდის ფრაგმენტის გარეშე ბრაუზერმა შეიძლება ავტომატურად გადართოს HTML -ის დოკუმენტის სხვა ვერსიაზე, რის შემდეგაც დაირღვევა საიტის ვიზუალური სტრუქტურა

  • < html lang="en">

    • html ტეგში მოტავსებული ატრიბუტი lang="en" ნიშნავს რომ გვერდი დაწერილია ინგლისურ ენაზე.

  • <head>

    • სექცია არის მეტამონაცემების და დამხმარე ინფორმაციისთვის, რომელიც გვერდზე არ ჩანს, მაგრამ გამოიყენება ბრაუზერის მიერ.
    • ამ სექციაში ხშირად შედის:
      a) < meta charset="UTF-8">
      აღნიშნავს დოკუმენტის კოდის ფორმატს. UTF-8 მხარს უჭერს მსოფლიოს სხვადასხვა ენას, რაც უზრუნველყოფს ტექსტის სწორ გამოტანას.

      b) < meta name="viewport" content="width=device-width, initial-scale=1.0">
      აყენებს პარამეტრებს სხვადასხვა მოწყობილობების ეკრანზე გამოსახვისთვის. width=device-width აყენებს გვერდის სიგანეს მოწყობილობის ეკრანის სიგანეზე. initial-scale=1.0 განსაზღვრავს გვერდის საწყის მასშტაბს.

      c) < title>Document < /title> განსაზღვრავს გვერდის სათაურს, რომელიც: ჩანს ბრაუზერის ფანჯრის თავში. ჩნდება საძიებო სისტემის შედეგებში.

  • < body>
    მთავარი ტეგი შიგთავსისთვის, რომელიც მომხმარებელს გვერდზე უჩანს. აქ მოთავსებულია ისეთი ელემენტები, როგორებიცაა ტექსტი, სურათები, ვიდეოები, ღილაკები, ბმულები და სხვა. ყველა ვიზუალური ელემენტი ამ ნაწილშია მოთავსებული.
